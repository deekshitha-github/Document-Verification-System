# 📄 Document Verification System

## Overview
The Document Verification System is an AI-powered application that verifies uploaded documents using OCR and deep learning techniques. It extracts text from documents, validates information, and provides accurate verification results.

---

## ✨ Features

- 📤 Upload document images or PDFs
- 🔍 OCR-based text extraction
- 🤖 Deep learning-based document verification
- 📊 Displays verification results
- 🖥️ User-friendly interface

---

## 🛠️ Tech Stack

- Python
- PyTorch
- OpenCV
- EasyOCR / Tesseract OCR
- Streamlit / Flask
- NumPy
- Pandas
